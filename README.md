# PRODIGY_WD_03

## Tic-Tac-Toe Web Application

Created as part of the **Prodigy InfoTech Web Development Internship**.

---

### Task Description

- Build a tic-tac-toe game using HTML, CSS, and JavaScript.
- Players can click cells to place X or O.
- Track game state and check for winning conditions.
- Highlight winning line (horizontal, vertical, diagonal).
- Include a reset button to start a new game.

---

### Technologies Used

- HTML5
- CSS3
- JavaScript

---

### Features

- Interactive X/O gameplay
- Highlights winning line
- Draw detection
- Reset functionality
- Responsive design for mobile
- Beginner-friendly, clean code
